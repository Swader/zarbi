# AI Agent On-chain

Extracted from
[Jarrod Watts' tutorial on Youtube](https://youtu.be/xjRqmy6p1-c) and turned
into a simple Deno runnable example.

Run it with:

```bash
deno run --allow-env --allow-net --allow-read src/main.ts
```
